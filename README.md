#  Calculadora de Médias com Validação de Dados

Este projeto é um sistema em Java desenvolvido para calcular a média aritmética de dois valores, garantindo que as entradas estejam dentro de um intervalo específico (1 a 10).

### 🛠️ Funcionalidades
* **Entrada de Dados:** Recebe notas do tipo `double` (permitindo casas decimais).
* **Validação de Intervalo:** Utiliza estruturas `while` para impedir que o programa aceite notas menores que 1 ou maiores que 10.
* **Cálculo de Média:** Processamento aritmético automático.
* **Sistema de Status:** * **Aprovado:** Média >= 8
  * **Recuperação:** Média >= 6 e < 8
  * **Reprovado:** Média < 6

### 💻 Conceitos Aplicados
1. **Lógica de Repetição:** Loops para garantir que o usuário forneça dados válidos antes de prosseguir.
2. **Operadores Lógicos:** Uso de `||` (OU) para validar faixas de valores.
3. **Estruturas Condicionais:** `if`, `else if` e `else` para determinar o resultado final.

### 🚀 Como executar
Basta clonar o repositório e executar a classe `Main.java` em qualquer IDE Java (recomendado IntelliJ IDEA).
